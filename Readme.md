#A Coding Count Tool

An easy tool for counting code

##Condition

 - required python2.7

##Usage

 - to analysis current path
 ```shell
python2.7 code_count.py
 ```

 - to analysis other path
 ```shell
python2.7 code_count.py <path>

 ```
